# Hoxo by Mushee

A dark, premium Next.js landing page and app preview for **Hoxo**, using the supplied logo and a padlock icon on every **Open App** button.

## Included

- Landing page at `/`
- App preview page at `/app`
- User-provided logo in `public/hoxo-logo.jpeg`
- Clean Next.js 14 setup ready for Vercel

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy to Vercel

### Option 1: GitHub

1. Unzip the folder
2. Push it to a GitHub repo
3. Import the repo into Vercel
4. Deploy

### Option 2: Vercel CLI

```bash
npm install -g vercel
vercel
```
